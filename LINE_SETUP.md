# 📲 วิธีตั้งค่าส่งสรุปเข้า LINE

> หมายเหตุ: LINE Notify ปิดบริการแล้ว (มี.ค. 2025) เราจึงใช้ **LINE Messaging API** ผ่าน **LINE Official Account (OA)** ซึ่งส่งข้อความฟรีได้จำนวนหนึ่งต่อเดือน

ตั้งค่าครั้งเดียว ใช้เวลาประมาณ 10 นาที มี 4 ขั้นตอน

---

## ขั้นที่ 1 — สร้าง LINE Official Account + Messaging API

1. เข้า https://developers.line.biz/console/ แล้ว Login ด้วยบัญชี LINE
2. สร้าง **Provider** ใหม่ (ตั้งชื่ออะไรก็ได้ เช่น ชื่อร้าน)
3. ในนั้นกด **Create a Messaging API channel** → กรอกข้อมูลพื้นฐาน (ชื่อ OA, หมวดหมู่) แล้วสร้าง
   - หากระบบให้สร้างผ่าน LINE Official Account Manager ก็ทำตามขั้นตอนจนได้ช่อง Messaging API

## ขั้นที่ 2 — เอา Channel Access Token

1. เปิด channel ที่สร้าง → แท็บ **Messaging API**
2. หา **Channel access token (long-lived)** → กด **Issue** เพื่อสร้าง แล้ว **คัดลอกไว้** (เป็นความลับ อย่าเผยแพร่)

## ขั้นที่ 3 — ใส่ Token ใน Render

1. เข้า Render → service **cat-food-order** → เมนู **Environment**
2. กด **Add Environment Variable** เพิ่ม:
   - Key: `LINE_TOKEN`  → Value: (วาง token ที่คัดลอกมา)
   - (ไม่บังคับแต่แนะนำ) Key: `LINE_SECRET` → Value: Channel secret (อยู่ในแท็บ Basic settings) เพื่อความปลอดภัยของ webhook
3. กด **Save** — Render จะ deploy ใหม่ให้อัตโนมัติ

> 🔒 token นี้เป็นความลับ ควรเป็นคุณวางเองในช่อง Environment ของ Render

## ขั้นที่ 4 — เชื่อม webhook + ปลายทาง

1. กลับไปที่ LINE Developers → channel → แท็บ **Messaging API**
2. ช่อง **Webhook URL** ใส่:
   ```
   https://cat-food-order.onrender.com/line/webhook
   ```
   แล้วกด **Update** และเปิด **Use webhook** ให้เป็น ON
3. เลือกปลายทางที่จะรับสรุป:
   - **ส่งเข้าแชทส่วนตัว:** สแกน QR ของ OA (อยู่ในแท็บ Messaging API) เพิ่มเป็นเพื่อน แล้ว **ทักข้อความอะไรก็ได้ 1 ครั้ง**
   - **ส่งเข้ากลุ่ม:** เชิญ OA เข้ากลุ่ม แล้วพิมพ์อะไรก็ได้ในกลุ่ม 1 ครั้ง
4. ถ้าเชื่อมสำเร็จ บอทจะตอบกลับว่า **"✅ เชื่อมต่อ ออเดอร์เหมียว เรียบร้อย!"** — ระบบจะจำปลายทางนี้อัตโนมัติ

## เสร็จแล้ว! 🎉
เข้าหลังบ้าน → แท็บ **ออเดอร์** → กดปุ่ม **📲 ส่งสรุปเข้าไลน์** ระบบจะส่งสรุปออเดอร์ปัจจุบันเข้าไลน์ทันที

ถ้าปุ่มขึ้นเตือนสีจาง = ยังตั้งค่าไม่ครบ ระบบจะบอกว่าติดตรงไหน (ยังไม่ใส่ token / ยังไม่ทักเชื่อมปลายทาง)
