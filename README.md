# 🐱 ออเดอร์เหมียว — ระบบจัดการออเดอร์อาหาร

ระบบสั่งอาหารแบบแชร์ข้ามเครื่อง (คอม + มือถือ) มีเซิร์ฟเวอร์กลางเก็บข้อมูล แบ่งเป็น **หน้าบ้าน** กับ **หลังบ้าน**

## หน้าเว็บ
- **หน้าบ้าน** (`/`) — สำหรับลูกค้า/เพื่อนร่วมงาน **สั่งได้อย่างเดียว** เลือกเมนูจากร้าน ใส่ชื่อ กดสั่ง เห็นรายการรวมทุกคน
- **หลังบ้าน** (`/admin.html`) — **แก้ไขได้ทุกอย่าง** เพิ่ม/แก้/ลบร้านค้า เมนู รูป ราคา เบอร์โทร และจัดการออเดอร์ + ดูสรุป

ทุกหน้าใช้งานได้ทั้งบนคอมและมือถือ (responsive) ข้อมูลอัปเดตอัตโนมัติทุก 5 วินาที

## รันบนเครื่องตัวเอง (ทดสอบ)
```
npm install
npm start
```
แล้วเปิด http://localhost:3000 (หน้าสั่ง) และ http://localhost:3000/admin.html (หลังบ้าน)

## เอาขึ้นออนไลน์
ดูขั้นตอนละเอียดในไฟล์ **DEPLOY.md**

## โครงสร้าง
```
server.js          เซิร์ฟเวอร์ + REST API
package.json       รายการ dependency
public/
  index.html       หน้าบ้าน (สั่งอาหาร)
  admin.html       หลังบ้าน (จัดการ)
  style.css        ธีมแมวพาสเทล
```

## หมายเหตุ
- ข้อมูลเก็บในไฟล์ `data.json` — บนโฮสต์ต้องตั้งค่าให้เก็บถาวร (ดู DEPLOY.md)
- ระบบนี้ **ไม่มีรหัสผ่าน** ใครมีลิงก์ `/admin.html` ก็เข้าหลังบ้านได้ ถ้าต้องการล็อกภายหลังบอกได้
